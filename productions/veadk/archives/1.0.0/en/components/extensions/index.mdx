---
title: "Overview"
---

Extensions are optional modules separate from the core capabilities, used to
connect VeADK agents to external channels, add safeguards, and similar. Import
them as needed.

| Extension | Purpose |
| --- | --- |
| Feishu | Bridges a Feishu bot with a `Runner`, so an agent can send and receive messages directly in Feishu conversations |
| Safeguards | Reviews content at each stage via callbacks and blocks unsafe input and output |
