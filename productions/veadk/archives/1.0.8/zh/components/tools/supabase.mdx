## 功能说明

`build_supabase_mcptoolset`（`from veadk.tools.builtin_tools.supabase_toolset import build_supabase_mcptoolset`）用于快速构建访问 [Supabase](https://supabase.com/) MCP Server 的工具集，接受 `url` 与 `api_key` 两个参数，返回一个可直接注册到智能体的 `MCPToolset` 实例。

## 使用方法

```python
from veadk import Agent
from veadk.tools.builtin_tools.supabase_toolset import build_supabase_mcptoolset

supabase_toolset = build_supabase_mcptoolset(
    url="https://your-supabase-mcp-endpoint",
    api_key="your-api-key-here",
)

agent = Agent(tools=[supabase_toolset])
```
