---
title: "可信 MCP"
---

可信 MCP（Trusted MCP）在标准 MCP 协议之上增加组件间的身份证明与验证能力，并提供端到端加密通信。结合机密计算（如 Jeddak AICC）与可信推理服务，可有效防范服务身份不可信、数据被篡改、流量劫持与隐私泄露等风险。

## 可信智能体

依托机密计算技术，可以将智能体与大模型服务（如机密豆包）运行在可信环境中，再通过可信 MCP 的端到端加密通信，构建全链路可信的智能体。VeADK 原生支持连接可信 MCP 服务，在工具调用链中开启可信通道，确保出站通信安全。

## 前提

<Steps>
  <Step title="准备可信 MCP 服务">
    获取可用的可信 MCP 服务地址，例如 `wss://your-trusted-mcp.example.com`。
  </Step>
  <Step title="准备 AICC 配置">
    准备 AICC 配置文件 `./aicc_config.json`（用于机密环境与远端证明），参考[官方示例](https://github.com/volcengine/AICC-Trusted-MCP/blob/main/README.md)。
  </Step>
</Steps>

## 使用方法

在连接参数中开启 `x-trusted-mcp: true`，再用 `TrustedMcpToolset` 接入：

```python title="agent.py"
import asyncio

from veadk import Agent
from veadk.utils.mcp_utils import get_mcp_params
from veadk.tools.mcp_tool.trusted_mcp_toolset import TrustedMcpToolset

mcp_url = "<可信 MCP 服务地址>"

# 开启可信通道
connection_params = get_mcp_params(mcp_url)
connection_params.headers = {"x-trusted-mcp": "true"}

# 初始化可信工具集并挂到智能体上
toolset = TrustedMcpToolset(connection_params=connection_params)
agent = Agent(tools=[toolset])

response = asyncio.run(agent.run("北京天气怎么样？"))
print(response)
```

## 配置项

`TrustedMcpToolset` 的关键配置：

| 配置 | 位置 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x-trusted-mcp` | Header | `true` | 是否开启可信 MCP 并启用可信通道 |
| `aicc_config` | 文件路径 | `./aicc_config.json` | AICC 配置文件路径，用于机密计算环境的证明与策略配置 |

配置文件的示例与说明见 [可信 MCP 配置文件](https://github.com/volcengine/AICC-Trusted-MCP/blob/main/README.md)。
