VeADK is a Python framework for building agent applications. It uses Google ADK's agent and execution model and integrates Volcengine models, storage, tools, security, and observability services for development from local prototypes to cloud applications.

## Core capabilities

- **Agent configuration**: configure models, system prompts, structured output, runtimes, and skills. Version 1.0.1 supports local and cloud skills through ADK `SkillToolset`.
- **Context and data**: manage conversational state and external knowledge through sessions, long-term memory, and knowledge bases.
- **Tool use**: extend an agent with functions, MCP servers, and built-in Volcengine tools.
- **Security and observability**: configure inbound and outbound authentication, content safety, trusted MCP, and trace exporters.
- **User interfaces**: debug with VeADK Web or build interactive experiences with A2UI and VeADK Frontend.
