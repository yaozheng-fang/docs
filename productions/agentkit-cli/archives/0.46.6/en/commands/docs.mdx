---
title: "View the docs"
---

`agentkit docs` opens the AgentKit CLI online documentation in your default browser. When no browser can be launched (for example a headless or continuous-integration environment), it prints the documentation URL in the terminal instead.

| Flag / argument | Description | Default |
| --- | --- | --- |
| `--print` | Print the documentation URL instead of opening a browser | `false` |

```bash
agentkit docs
agentkit docs --print
```
