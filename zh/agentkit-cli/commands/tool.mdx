title: "管理工具"
description: "管理 AgentKit 工具。"
---

`tool` 命令组用于管理 AgentKit 工具，涵盖列表、详情查看、创建、更新与删除等操作。

## tool list

列出所有工具。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `-r, --region <region>` | Volcengine 区域。 | 自动检测 |
| `-p, --project <name>` | 项目名称。 | — |
| `--type <type>` | 按工具类型过滤：`mcp` \| `code` \| `skill`。 | — |
| `--json` | 输出原始 JSON。 | `false` |

```bash
agentkit tool list --type mcp
```

## tool show

查看工具的详情。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `<id>` | 工具 id，例如 `t-xxxxxxxx`。必填。 | — |
| `-r, --region <region>` | Volcengine 区域。 | 自动检测 |
| `--json` | 输出原始 JSON。 | `false` |

```bash
agentkit tool show t-12345678
```

## tool create

创建工具。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `--name <name>` | 工具名称。 | — |
| `--tool-type <type>` | 工具类型：`mcp` \| `code` \| `skill`。 | — |
| `-r, --region <region>` | Volcengine 区域。 | 来自环境变量 |
| `--description <text>` | 工具描述。 | — |
| `--command <command>` | 容器启动命令。 | — |
| `--image-url <url>` | 容器镜像地址。 | — |
| `--port <port>` | 容器端口。 | — |
| `-p, --project <name>` | 项目名称。 | — |
| `--role-name <role>` | IAM 角色名称。 | — |
| `--apmplus` | 启用 APMPlus。 | — |
| `--cpu <milli>` | CPU，单位为毫核（沙箱）。 | — |
| `--memory <mb>` | 内存，单位为 MB（沙箱）。 | — |
| `--enable-security` | 启用安全沙箱。 | — |
| `--enable-tos` | 将 TOS 挂载进沙箱。 | — |
| `--env <KEY=VALUE>` | 环境变量（可重复）。 | — |
| `--json <jsonString>` | 以 JSON 对象形式提供的额外字段，合并进请求体。 | — |

```bash
agentkit tool create --name my-tool --tool-type code --image-url registry/my-tool:latest --port 8080
```

## tool update

更新工具。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `<id>` | 工具 id，例如 `t-xxxxxxxx`。必填。 | — |
| `-r, --region <region>` | Volcengine 区域。 | 自动检测 |
| `--tool-type <type>` | 工具类型。 | — |
| `--description <text>` | 工具描述。 | — |
| `--command <command>` | 容器启动命令。 | — |
| `--image-url <url>` | 容器镜像地址。 | — |
| `--port <port>` | 容器端口。 | — |
| `--apmplus` | 启用 APMPlus。 | — |
| `--json <jsonString>` | 以 JSON 对象形式提供的额外字段，合并进请求体。 | — |

```bash
agentkit tool update t-12345678 --description "更新后的工具" --port 9090
```

## tool delete

删除工具。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `<id>` | 工具 id，例如 `t-xxxxxxxx`。必填。 | — |
| `-r, --region <region>` | Volcengine 区域。 | 自动检测 |
| `-y, --yes` | 跳过确认提示。 | `false` |

```bash
agentkit tool delete t-12345678 --yes
```
