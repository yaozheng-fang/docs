---
title: "管理平台技能"
---

`skill` 命令组用于管理 AgentKit 技能平台上的技能：列出、查看详情、查看版本、查看技能空间，以及删除技能。

## skill list

列出技能。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `-r, --region <region>` | Volcengine 区域 | 无 |
| `-p, --project <name>` | 项目名称 | 无 |
| `--json` | 输出原始 JSON | `false` |

```bash
agentkit skill list
```

## skill show

查看某个技能的详情。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `<id>` | 技能 ID（必填） | 无 |
| `-r, --region <region>` | Volcengine 区域 | 无 |
| `--json` | 输出原始 JSON | `false` |

```bash
agentkit skill show sk-xxxxxxxx
```
