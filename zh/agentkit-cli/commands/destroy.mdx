---
title: "销毁运行时"
---

`destroy` 删除已部署的运行时。不带名称时，从项目本地配置（`.agentkit/agentkit.yaml`）解析目标运行时；也可传入名称以显式指定。该命令删除运行时本身，用于拆除不再需要的部署。

| 标志/参数 | 说明 | 默认值 |
| --- | --- | --- |
| `[name]` | 要删除的运行时名称。省略则使用项目本地配置。 | 取自配置 |
| `-r, --region <region>` | Volcengine 区域。 | 自动探测 |
| `-y, --yes` | 跳过确认提示。 | `false` |

```bash
# 删除当前项目配置中声明的运行时
agentkit destroy

# 按名称删除指定运行时，且不弹出确认
agentkit destroy my-agent --yes
```
