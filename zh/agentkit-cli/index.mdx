`agentkit` 是 AgentKit 的命令行工具：脚手架项目、构建与部署运行时（runtime），
并管理工具（tool）、知识库（knowledge）、记忆（memory）和评测（evaluation）——全部在终端完成。

## 命令分组

| 分组 | 管理内容 |
| --- | --- |
| `init` | 从模板脚手架一个新项目 |
| `deploy` | 构建并部署项目（`config` → `build` → `apply`） |
| `runtime` | 已部署的运行时（list/show/logs/versions/release/update/delete） |
| `knowledge` (`kb`) | 知识库 |
| `memory` (`mem`) | 记忆集合 |
| `tool` | 工具（MCP server、代码环境等） |
| `harness` | 由 `harness.yaml` 定义的免代码 agent |
| `invoke` | 调用已部署的运行时 |
| `auth` | SSO 登录/登出/身份，以及登录 profile |
| `onboard` | 把内置技能装进本地 coding agent |
| `eval dataset` | 评测数据集及其用例（`eval` 前缀可省略） |
| `tree` | 打印完整命令树 |

## 约定

- `knowledge`↔`kb`、`memory`↔`mem` 为别名；`eval` 前缀可省略
  （`agentkit dataset …` 等价于 `agentkit eval dataset …`）。
- 大多数读命令支持 `--json`；region 默认自动探测，可用 `-r, --region` 指定。
- 破坏性命令（`delete`、`remove`）默认二次确认，`-y` 跳过。
