---
title: "提示词管理"
description: "在运行时模板化并热更新系统提示词"
---

在生产环境中，系统提示词往往需要版本管理与动态下发。提示词管理让你在运行时模板化并热更新提示词，而无需修改代码或重启服务。每次处理用户请求前，VeADK 都会调用提示词管理器的 `get_prompt` 方法获取最新内容，并将其作为该智能体的系统提示词。

## 使用 CozeLoop

VeADK 内置 `CozeloopPromptManager`，可将系统提示词对接到 CozeLoop 云端提示词管理。

先安装 CozeLoop 的 Python SDK：

```bash
pip install cozeloop
```

将凭证通过环境变量提供，避免在代码中硬编码：

```bash
export COZELOOP_WORKSPACE_ID="<你的 Workspace ID>"
export COZELOOP_TOKEN="<你的 CozeLoop Token>"
```

创建管理器并传入智能体：

```python
import os
from veadk import Agent
from veadk.prompts.prompt_manager import CozeloopPromptManager

prompt_manager = CozeloopPromptManager(
    cozeloop_workspace_id=os.environ["COZELOOP_WORKSPACE_ID"],
    cozeloop_token=os.environ["COZELOOP_TOKEN"],
    prompt_key="your_prompt_key",  # CozeLoop 中创建的 Prompt Key
    label="production",            # CozeLoop 中创建的 Prompt 标签
)

agent = Agent(name="assistant", prompt_manager=prompt_manager)
```

<Note>
  CozeLoop 会在本地缓存提示词，更新间隔为 1 分钟。当获取失败时，会回退到 VeADK 默认的系统提示词。
</Note>

## 自定义提示词管理器

若需从自有的配置中心、数据库或文件加载提示词，可继承 `BasePromptManager` 并实现 `get_prompt` 方法。该方法在每次处理用户请求前被调用，返回的字符串将作为该智能体的系统提示词。
