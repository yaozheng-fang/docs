VeADK 是用于构建、运行和部署智能体应用的 Python 开发套件，提供模型、工具、记忆、知识库、可观测、安全与云服务集成。本目录对应 VeADK `1.0.4`，页面中的公开 API、命令和默认行为均以该版本为准。
