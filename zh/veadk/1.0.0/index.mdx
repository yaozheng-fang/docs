---
title: "概述"
---

VeADK 是面向智能体应用开发的 Python 框架。它基于 Google ADK 的智能体与运行机制，提供火山引擎模型、存储、工具、安全和可观测服务的集成，适用于从本地原型到云端应用的开发过程。

## 核心能力

- **智能体配置**：设置模型、系统提示词、结构化输出、运行时和技能。
- **上下文与数据**：通过会话、长期记忆和知识库管理对话状态与外部知识。
- **工具调用**：使用函数、MCP Server 以及火山引擎内置工具扩展智能体能力。
- **安全与可观测性**：配置入站与出站认证、内容安全、可信 MCP 和调用链导出。
- **交互界面**：使用 VeADK Web 调试智能体，或通过 A2UI 和 VeADK Frontend 构建交互界面。

## 开始使用

先完成[安装](/zh/veadk/1.0.0/get-started/installation)，再按照[快速开始](/zh/veadk/1.0.0/get-started/quickstart)运行最小智能体。需要查询版本变化时，请参见 [1.0.0 变更记录](/zh/veadk/1.0.0/changelog)。
