本教程介绍如何安装 VeADK、配置模型，并运行一个最小可用的 agent，最终在终端获得模型的回答。

<Note>
  本章仅覆盖从安装到首次运行的最小路径。部署到 AgentKit、记忆、知识库、多 agent、评测等主题请参见对应章节。
</Note>

## 前置条件

- Python 3.10 及以上版本。
- 火山方舟（Ark）API Key，用于调用模型，可在[火山引擎方舟控制台](https://console.volcengine.com/ark)创建。

## 安装

从 PyPI 安装核心包：

```bash
pip install veadk-python
```

VeADK 通过可选依赖组（extras）提供按需能力，可根据用途单独安装：

```bash
pip install "veadk-python[extensions]"
```

可用的依赖组：

| Extra | 用途 |
| --- | --- |
| `extensions` | 飞书渠道、Cozeloop、LlamaIndex 等集成 |
| `database` | Redis、MySQL、Viking DB、mem0 等存储后端 |
| `eval` | 基于 DeepEval 与 Google ADK 的评测 |
| `a2ui` | A2UI（agent 驱动的界面） |
| `pdf` | PDF 页面渲染 |

## 配置模型

在项目根目录创建 `config.yaml`，VeADK 会自动读取。运行最小 agent 仅需配置模型：

```yaml config.yaml
model:
  agent:
    provider: openai
    name: doubao-seed-1-6-250615
    api_base: https://ark.cn-beijing.volces.com/api/v3/
    api_key: # 填入火山方舟 API Key
```

<Tip>
  `provider: openai` 表示采用 OpenAI 兼容协议，`api_base` 指向火山方舟的兼容接口，因此可直接使用豆包（doubao）等方舟模型。
</Tip>

## 编写第一个 agent

创建 `main.py`：

```python main.py
from veadk import Agent
import asyncio

agent = Agent()

res = asyncio.run(agent.run("用一句话介绍火山引擎（Volcengine）。"))
print(res)
```

`Agent` 持有模型与指令，不传参数时使用 `config.yaml` 中的默认模型；`agent.run(...)` 为异步方法，返回最终的文本回答。

运行：

```bash
python main.py
```

终端将打印模型返回的回答。至此，一个最小 VeADK agent 已完成运行。
