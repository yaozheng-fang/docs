This tutorial explains how to install VeADK, configure a model, and run a
minimal agent that returns an answer in your terminal.

<Note>
  This chapter covers only the minimal path from installation to a first run.
  Deploying to AgentKit, memory, knowledge bases, multi-agent, and evaluation
  are covered in their respective chapters.
</Note>

## Prerequisites

- Python 3.10 or later.
- A Volcengine Ark API key for calling models, created in the [Ark console](https://console.volcengine.com/ark).

## Installation

Install the core package from [PyPI](https://pypi.org/project/veadk-python/):

```bash
pip install veadk-python
```

VeADK provides optional dependency groups (extras) for on-demand capabilities,
installed individually as needed:

```bash
pip install "veadk-python[extensions]"
```

Available extras:

| Extra | Purpose |
| --- | --- |
| `extensions` | Feishu channel, Cozeloop, LlamaIndex, and related integrations |
| `database` | Redis, MySQL, Viking DB, mem0, and other storage backends |
| `eval` | Evaluation via DeepEval and Google ADK |
| `a2ui` | A2UI (agent-driven UI) |
| `pdf` | PDF page rendering |

## Configure a model

Create `config.yaml` in the project root; VeADK reads it automatically. A
minimal agent only requires the model to be configured:

```yaml config.yaml
model:
  agent:
    provider: openai
    name: doubao-seed-1-6-250615
    api_base: https://ark.cn-beijing.volces.com/api/v3/
    api_key: # your Volcengine Ark API key
```

<Tip>
  `provider: openai` uses the OpenAI-compatible protocol, and `api_base` points
  at Ark's compatible endpoint, so Ark models such as doubao can be used directly.
</Tip>

## Write your first agent

Create `main.py`:

```python main.py
from veadk import Agent
import asyncio

agent = Agent()

res = asyncio.run(agent.run("Introduce Volcengine in one sentence."))
print(res)
```

`Agent` holds the model and instruction; with no arguments it uses the default
model from `config.yaml`. `agent.run(...)` is asynchronous and returns the final
text answer.

Run it:

```bash
python main.py
```

The terminal prints the model's answer, completing a minimal VeADK agent run.
