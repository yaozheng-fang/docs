## Overview

`mcp_router` (`from veadk.tools.builtin_tools.mcp_router import mcp_router`) aggregates and forwards the tools of multiple MCP Servers through the Volcengine MCP Router, which is ideal when you need a single entry point for several MCP services. It works over a streamable HTTP connection (`StreamableHTTPConnectionParams`).

## Environment & prerequisites

Environment variables:

- `TOOL_MCP_ROUTER_URL`: the MCP Router service address
- `TOOL_MCP_ROUTER_API_KEY`: the API key for authentication, sent as the `Authorization: Bearer <key>` request header

Or configure in `config.yaml`:

```yaml title="config.yaml"
tool:
  mcp_router:
    url: https://your-mcp-router-endpoint
    api_key: your-api-key-here
```

## Usage

```python
from veadk import Agent
from veadk.tools.builtin_tools.mcp_router import mcp_router

agent = Agent(tools=[mcp_router])
```
