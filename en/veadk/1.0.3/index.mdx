VeADK is a Python framework for developing agent applications. It is compatible with Google ADK's agent and runtime model and integrates Volcengine models, storage, tools, security, and observability. Public APIs, configuration, and defaults in this section follow the `1.0.3` tag.

## Core capabilities

- **Agent configuration**: models, instructions, structured output, ADK/Codex runtimes, and skills.
- **Context and data**: short-term sessions, long-term memory, and knowledge bases, with Milvus and OpenViking added in 1.0.3.
- **Tool use**: functions, MCP servers, and Volcengine tools.
- **Security and identity propagation**: inbound and outbound authentication, including TIP Token or Bearer JWT propagation across A2A calls.
- **Observability and interfaces**: full trace export plus VeADK Web and VeADK Frontend for debugging.
