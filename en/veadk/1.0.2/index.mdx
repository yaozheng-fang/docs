VeADK is a Python framework for developing agent applications. It is compatible with Google ADK's agent and runtime model and integrates Volcengine models, storage, tools, security, and observability. Public APIs, configuration, and defaults in this section follow the `1.0.2` tag.

## Core capabilities

- **Agent configuration**: models, instructions, structured output, ADK/Codex runtimes, skills, and Ark API-key lookup by name.
- **Context and data**: short-term sessions, long-term memory, and knowledge bases, including PostgreSQL schema isolation.
- **Tool use**: functions, MCP servers, and Volcengine tools, with one automatic reconnect after an MCP connection drops.
- **Security and observability**: inbound and outbound authentication, content safety, trusted MCP, and trace export.
- **User interfaces**: VeADK Web and VeADK Frontend for debugging and MCP OAuth flows.
