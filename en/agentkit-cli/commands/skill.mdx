---
title: "Manage platform skills"
description: "Manage skills on the AgentKit skill platform."
---

The `skill` command group manages skills on the AgentKit skill platform: list, show details, list versions, list skill spaces, and delete skills.

## skill list

List skills.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `-r, --region <region>` | Volcengine region | None |
| `-p, --project <name>` | Project name | None |
| `--json` | Output raw JSON | `false` |

```bash
agentkit skill list
```

## skill show

Show a skill's details.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Skill id (required) | None |
| `-r, --region <region>` | Volcengine region | None |
| `--json` | Output raw JSON | `false` |

```bash
agentkit skill show sk-xxxxxxxx
```
