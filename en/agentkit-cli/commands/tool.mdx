---
title: "Manage tools"
---

The `tool` command group manages AgentKit tools, covering listing, inspection, creation, update, and deletion.

## tool list

List all tools.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `-p, --project <name>` | Project name. | — |
| `--type <type>` | Filter by tool type: `mcp` \| `code` \| `skill`. | — |
| `--json` | Output raw JSON. | `false` |

```bash
agentkit tool list --type mcp
```

## tool show

Show details of a tool.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Tool id, e.g. `t-xxxxxxxx`. Required. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--json` | Output raw JSON. | `false` |

```bash
agentkit tool show t-12345678
```

## tool create

Create a tool.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `--name <name>` | Tool name. | — |
| `--tool-type <type>` | Tool type: `mcp` \| `code` \| `skill`. | — |
| `-r, --region <region>` | Volcengine region. | from env |
| `--description <text>` | Tool description. | — |
| `--command <command>` | Container start command. | — |
| `--image-url <url>` | Container image url. | — |
| `--port <port>` | Container port. | — |
| `-p, --project <name>` | Project name. | — |
| `--role-name <role>` | IAM role name. | — |
| `--apmplus` | Enable APMPlus. | — |
| `--cpu <milli>` | CPU in milli-cores (sandbox). | — |
| `--memory <mb>` | Memory in MB (sandbox). | — |
| `--enable-security` | Enable the security sandbox. | — |
| `--enable-tos` | Mount TOS into the sandbox. | — |
| `--env <KEY=VALUE>` | Environment variable (repeatable). | — |
| `--json <jsonString>` | Extra fields as a JSON object, merged into the request body. | — |

```bash
agentkit tool create --name my-tool --tool-type code --image-url registry/my-tool:latest --port 8080
```

## tool update

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Tool id, e.g. `t-xxxxxxxx`. Required. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--tool-type <type>` | Tool type. | — |
| `--description <text>` | Tool description. | — |

| `--image-url <url>` | Container image url. | — |
| `--port <port>` | Container port. | — |

| `--json <jsonString>` | Extra fields as a JSON object, merged into the request body. | — |


## tool delete

Delete a tool.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Tool id, e.g. `t-xxxxxxxx`. Required. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `-y, --yes` | Skip the confirmation prompt. | `false` |

```bash
agentkit tool delete t-12345678 --yes
```
