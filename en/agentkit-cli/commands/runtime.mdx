---
title: "Manage runtimes"
description: "Manage AgentKit runtimes."
---

The `runtime` command group manages AgentKit runtimes, covering listing, inspection, log retrieval, version history, releasing, updating, and deletion.

## runtime list

List all runtimes.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `-p, --project <name>` | Project name. | — |
| `--json` | Output raw JSON. | `false` |

```bash
agentkit runtime list --project demo
```

## runtime show

Show details of a runtime, optionally a specific version.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<runtime>` | Runtime id (`r-...`) or name. Required. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--rev <n>` | Show a specific version instead of the current one. | current |
| `--json` | Output raw JSON. | `false` |

```bash
agentkit runtime show my-agent --rev 3
```

## runtime delete

Delete a runtime.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<runtime>` | Runtime id (`r-...`) or name. Required. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `-y, --yes` | Skip the confirmation prompt. | `false` |

```bash
agentkit runtime delete my-agent --yes
```

## runtime logs

Show logs for a runtime instance.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<runtime>` | Runtime id (`r-...`) or name. Required. | — |
| `-i, --instance <name>` | Instance name. | auto-resolve |
| `-n, --limit <n>` | Number of log lines; `0` = all. | server-side |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--json` | Output raw JSON. | `false` |

```bash
agentkit runtime logs my-agent --limit 200
```

## runtime versions

List a runtime's version history.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<runtime>` | Runtime id (`r-...`) or name. Required. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--json` | Output raw JSON. | `false` |

```bash
agentkit runtime versions my-agent
```

## runtime release

Release (publish) a runtime version, making it live.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<runtime>` | Runtime id (`r-...`) or name. Required. | — |
| `--rev <n>` | Version number to release. | latest |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--no-wait` | Return immediately instead of waiting for `Ready`. | `false` |

```bash
agentkit runtime release my-agent --rev 3
```

## runtime update

Update a runtime's configuration. This does not release the new version unless `--auto-release` is passed.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<runtime>` | Runtime id (`r-...`) or name. Required. | — |
| `--description <text>` | Description. | — |
| `--artifact-url <url>` | Container image url. | — |
| `--artifact-type <type>` | Artifact type. | — |
| `--cpu-milli <n>` | CPU in milli-cores. | — |
| `--memory-mb <n>` | Memory in MB. | — |
| `--min-instance <n>` | Minimum instances. | — |
| `--max-instance <n>` | Maximum instances. | — |
| `--max-concurrency <n>` | Maximum concurrency per instance. | — |
| `--model-agent-name <name>` | Model agent name. | — |
| `--knowledge-id <id>` | Associated knowledge base id. | — |
| `--memory-id <id>` | Associated memory collection id. | — |
| `--tool-id <id>` | Associated tool id. | — |
| `--mcp-toolset-id <id>` | Associated MCP toolset id. | — |
| `--apmplus` | Enable APMPlus monitoring. | — |
| `--no-apmplus` | Disable APMPlus monitoring. | — |
| `--envs-json <json>` | Env vars as a JSON array, e.g. `'[{"Key":"K","Value":"V"}]'`. | — |
| `--tags-json <json>` | Tags as a JSON array. | — |
| `--json <jsonString>` | Advanced: JSON object merged into the request body. | — |
| `-r, --region <region>` | Volcengine region. | auto-detect |
| `--auto-release` | Release the new version immediately after updating. | `false` |
| `--no-wait` | Return immediately instead of waiting for the runtime to settle. | `false` |

```bash
agentkit runtime update my-agent --cpu-milli 1000 --memory-mb 2048 --auto-release
```
