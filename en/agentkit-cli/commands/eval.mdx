---
title: "Manage evaluation datasets"
description: "Manage evaluation datasets."
---

The `eval dataset` command group manages evaluation datasets: list, show, create, and delete datasets, and add or remove cases within a dataset.

<Note>
These subcommands are also available as top-level aliases, so the `eval` prefix may be omitted: `agentkit eval dataset list` is equivalent to `agentkit dataset list`. The examples below use the shorter `agentkit dataset ...` form.
</Note>

## dataset list

List evaluation datasets.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `-r, --region <region>` | Volcengine region | Auto-detect |
| `-p, --project <name>` | Project name | None |
| `--json` | Output raw JSON | `false` |

```bash
agentkit dataset list
```

## dataset show

Show details of an evaluation dataset, including its case items.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Evaluation set id or name (required) | None |
| `-r, --region <region>` | Volcengine region | Auto-detect |
| `-p, --project <name>` | Project name | `default` |
| `--items <n>` | List up to N case items (0 to skip) | `20` |
| `--json` | Output raw JSON | `false` |

```bash
agentkit dataset show my-dataset --items 50
```

## dataset create

Create an evaluation dataset.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `--name <name>` | Dataset name | None |
| `--schema <fields>` | Comma-separated field names | `input,reference_output,output` |
| `--description <text>` | Dataset description | None |
| `-r, --region <region>` | Volcengine region | From env |
| `-p, --project <name>` | Project name | `default` |
| `--json` | Output raw JSON | `false` |

```bash
agentkit dataset create --name my-dataset --schema "input,reference_output,output"
```

## dataset delete

Delete an evaluation dataset.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Evaluation set id or name (required) | None |
| `-r, --region <region>` | Volcengine region | Auto-detect |
| `-p, --project <name>` | Project name | `default` |
| `-y, --yes` | Skip the confirmation prompt | `false` |

```bash
agentkit dataset delete my-dataset -y
```

## dataset add

Add one or more cases to an evaluation dataset.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Evaluation set id or name (required) | None |
| `--field <key=value>` | Case field (repeatable) | None |
| `--file <path>` | JSON file: an object, or an array of `field: value` objects | None |
| `-r, --region <region>` | Volcengine region | Auto-detect |
| `-p, --project <name>` | Project name | `default` |
| `--json` | Output raw JSON | `false` |

```bash
agentkit dataset add my-dataset --field "input=Hello" --field "reference_output=Hi"
```

## dataset remove

Remove one or more cases from an evaluation dataset.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `<id>` | Evaluation set id or name (required) | None |
| `<itemIds...>` | Case item id(s) to remove (one or more, required) | None |
| `-r, --region <region>` | Volcengine region | Auto-detect |
| `-p, --project <name>` | Project name | `default` |
| `-y, --yes` | Skip the confirmation prompt | `false` |
| `--json` | Output raw JSON | `false` |

```bash
agentkit dataset remove my-dataset item-1 item-2 -y
```
