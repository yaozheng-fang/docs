---
title: "Delete a runtime"
---

The `destroy` command deletes the deployed runtime, either by name or from the local configuration.

## destroy

Delete the deployed runtime. When no name is supplied, the target is resolved from the local config.

| Flag / Argument | Description | Default |
| --- | --- | --- |
| `[name]` | Runtime name. | from local config |
| `-r, --region <region>` | Volcengine region. | auto-sense |
| `-y, --yes` | Skip confirmation. | `false` |

```bash
agentkit destroy my-agent --yes
```
